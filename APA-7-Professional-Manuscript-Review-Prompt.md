# APA 7 Student Paper Audit Protocol

## Command

**Run APA-STUDENT-AUDIT.**

## Purpose

When I type **“Run APA-STUDENT-AUDIT,”** review the attached student paper strictly for **APA 7 student paper style, citation, reference, and scholarly presentation compliance**.

Do **not** evaluate content quality, research design, argument strength, grading merit, or subject-matter accuracy unless I explicitly ask for that separately.

---

## Default APA Standard

Default to **APA 7 student paper format**, not professional manuscript format.

Do **not** require a running head unless the assignment, instructor, or institution specifically requires one.

Do **not** require an abstract unless the assignment directions require one.

---

## File Review Rules

Use the **Word-exported PDF as the visual authority** for layout issues, including:

- title page placement
- page numbers
- margins
- spacing
- paragraph indentation
- heading placement
- reference-list hanging indents
- tables
- figures
- page breaks

Use the **DOCX file to inspect structure**, including:

- paragraph settings
- heading levels
- citation patterns
- reference entries
- table and figure labels
- consistency across the paper

If only a DOCX is provided, render the DOCX to PDF/page images and clearly label layout findings as **engine-rendered and requiring confirmation in Microsoft Word**.

Do **not** guess when a layout issue cannot be verified.

---

## APA 7 Student Paper Items to Check

Check the following APA 7 student-paper elements:

- title page elements
- page number placement
- absence or presence of running head
- title formatting
- margins
- font consistency
- line spacing
- paragraph indentation
- heading levels
- capitalization
- italics
- abbreviation use
- number style
- in-text citations
- reference-list formatting
- hanging indents
- DOI/URL formatting
- table format
- figure format
- appendix formatting
- consistency across the document

---

## Student Title Page Requirements

For the student title page, check for the required student-paper elements:

- paper title
- author name
- institutional affiliation
- course number and name
- instructor name
- assignment due date
- page number

Flag missing elements unless the assignment instructions clearly indicate otherwise.

---

## Rubric Levels

Apply the following rubric and return only the **named level**, with no number, no percentage, and no letter grade.

### Exemplary

Follows all APA guidelines for in-text citations and the References page.

### Satisfactory

Follows most APA guidelines for in-text citations and the References page.

### Progressing

To some extent, the paper represents mature, scholarly, graduate-level writing, with APA generally followed.

### Insufficient

The quality of writing and/or APA formatting is not acceptable for graduate-level work.

### Incomplete

No meaningful attempt is present.

---

## Leveling Guidance

The named level should be based primarily on:

- APA 7 in-text citation accuracy
- References page accuracy
- graduate-level scholarly presentation

Consider layout and formatting issues when they materially affect APA compliance, but do not over-penalize minor isolated formatting issues.

---

## Required Response Structure

Return the review using the following structure:

## APA 7 Student Paper Level

**[Exemplary, Satisfactory, Progressing, Insufficient, or Incomplete]**

## Rationale

Briefly explain why the paper falls at that level, focusing on citation, references, APA style, and graduate-level scholarly presentation.

## Required Corrections

Provide a prioritized list of required corrections. For each issue, identify:

- page or section
- observed problem
- APA expectation
- required correction

## Repeated Pattern Issues

Identify recurring APA problems, especially repeated issues with:

- citations
- reference entries
- headings
- indentation
- title page elements
- spacing
- tables or figures

## Do Not Count Against APA Level Unless Required by Assignment

Do not count the following against the APA level unless the assignment, instructor, or institution specifically requires them:

- running head
- abstract
- professional-paper elements

---

## Restrictions

Do not provide:

- numeric score
- point value
- percentage
- letter grade

Do not overwhelm the student with minor preferences unless they recur or affect APA compliance.
